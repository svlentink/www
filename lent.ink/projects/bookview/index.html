Bookviewer

During college, you need to read books for exams. Sometimes a very expensive book has only one or two chapters you need to know.
Usually some student somewhere scans the pages and shares them.
In the past, people shared a folder with images (scanned pages), nowadays it's mostly PDF.
This tool was build before that age.
Browse Go through pages
list Compose list of chapters
This tool depends on a CDN which hosts the pictures. Usually this content is copy write protected.
Therefor we have the metadata of the book separate. 

{
  "pagesUrl": "http://shady.server.com/mybook/",
  "index":{
    "chapterid": ["title","alternative-title"]
  },
  "chapters":{
    "chapterid": {
      "start": 42, #number of starting page
      "end": 45
    }
  },
  "pages":{
    "42": {
      "firstChapterOnPage" : "chapterid", #optional
      "file": "042_mybook.jpg"
    }
  }
}
You supply your .json with this info as a hash parameter to the tools.
